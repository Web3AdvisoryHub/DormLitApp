## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   ```bash
   cp .env.example .env
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## Development

- Frontend: `cd client && npm run dev`
- Backend: `cd server && npm run dev`
- Build: `npm run build`
- Test: `npm run test`
